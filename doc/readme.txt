This project demonstrates the following

1. Vagrant vm
2. Puppet to install all software
3. Virtualenv
4. Python
5. IPython
6. Web based notebook for IPython
7. Using selenium to drive Firefox on a headless server and take screenshots

1. To use this project in the root type
vagrant up

2. Connect to the VM
vm\ssh_vagrant.bat

4. Start the notebook
/vagrant/vm/ipython_notebook.sh

5. Open the notebook in the browser at
http://localhost:8888/

6. The notebooks are stored in
/vagrant/notebooks
